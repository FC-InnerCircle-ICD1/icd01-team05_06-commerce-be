== 개요

이 문서는 커머스 서비스의 주문 관련 API 명세를 제공합니다. 개발 참고 용도로 작성되었습니다.

[[common-response]]
== 공통 응답 구조

모든 API 응답은 다음 구조를 따릅니다:

[source,json]
----
{
  "success": boolean,
  "data": object | null,
  "error": {
    "code": string,
    "message": string
  } | null
}
----
